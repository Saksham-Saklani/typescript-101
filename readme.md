# Typescript 101

TS -> Process -> JS

TS -> lexer -> binder -> checker -> emitter -> .js, .map, etc

## Setup

1. Install npm: `npm init -y`

2. Install TS: `npm i -D typescript`

3. Generate Config: `tsc --init`

4. Execute: `npx tsc`

### Set Config Dir

**rootDir**: TS files
**outDir**: JS files

### ttpe installation

1. `npm i library-name`

2. `npm i -D @types/library-name`
